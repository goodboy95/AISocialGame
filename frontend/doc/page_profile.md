# 个人中心页 (Profile Page)

**文件路径:** `src/pages/Profile.tsx`

## 1. 页面概述
展示当前登录用户的详细信息。

## 2. 关键元素
*   **头部卡片**: 包含头像、昵称、UID 和编辑按钮。背景使用品牌渐变色。
*   **数据概览**: 金币数、等级、游戏时长（三列布局）。
*   **详情 Tabs**:
    *   **对战记录**: 列表展示最近的胜负情况。
    *   **数据统计**: 使用进度条 (`Progress` 组件) 可视化展示胜率。

## 3. 开发要点
*   **UI 组件**: 大量使用了 shadcn/ui 的基础组件 (`Card`, `Avatar`, `Progress`, `Tabs`)。
