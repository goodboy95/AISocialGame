# 房间列表页 (Room List Page)

**文件路径:** `src/pages/RoomList.tsx`

## 1. 页面概述
用户选择特定游戏（如狼人杀）后进入此页面。这里列出了当前所有正在进行或等待中的房间，数据实时来自后端接口。

## 2. 关键元素

### 2.1 顶部操作栏
*   **返回按钮**: 返回首页。
*   **搜索框**: 允许通过房间号或名称筛选（目前为 UI 演示）。
*   **创建房间按钮**: 跳转到 `/create/:gameId`。

### 2.2 房间卡片 (Room Card)
*   **状态区分**:
    *   `waiting`: 显示绿色 Badge，按钮为“立即加入”。
    *   `playing`: 显示灰色 Badge，按钮为“观战”。
*   **信息展示**: 房间名、人数进度 (当前/最大)、标签 (Tags)。
*   **标签来源**: 基于房间属性动态生成，包括“公开/私密”、“语音/文字”、“玩家人数（xx人局）”以及板子模板标签（从 `template` 配置解析）。

## 3. 开发要点
*   **数据源**: 通过 React Query 调用 `roomApi.list(gameId)` 获取后端实时房间列表；`gameApi.detail` 补充模板标签映射。
*   **参数获取**: 使用 `useParams` 获取 `gameId`，从而从配置中读取游戏名称。
*   **状态映射**: 按钮样式和文本根据房间状态 (`status`) 动态变化。
