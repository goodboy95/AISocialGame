# 社区广场页 (Community Page)

**文件路径:** `src/pages/Community.tsx`

## 1. 页面概述
玩家交流中心，提供类似社交媒体的信息流体验。

## 2. 关键元素
*   **发帖框**: 顶部输入区域。
*   **内容 Tabs**: 推荐 / 最新 / 关注。
*   **帖子卡片**: 展示用户信息、内容、标签、点赞/评论数。
*   **侧边栏 (桌面端)**: 展示热门话题榜单。
*   **发布逻辑**: 输入框内容会即时追加到推荐流并触发成功提示（前端内存存储，便于无后端场景演示）。

## 3. 开发要点
*   **布局**: 典型的“左导航-中内容-右侧栏”布局（Holy Grail Layout 变体）。
*   **响应式**: 移动端隐藏左右侧边栏，仅保留核心信息流。
