# 首页 (Index Page)

**文件路径:** `src/pages/Index.tsx`

## 1. 页面概述
这是用户进入应用的第一站。主要目标是展示平台特色，并引导用户选择感兴趣的游戏进入大厅。

## 2. 关键元素

### 2.1 Hero Section (顶部横幅)
*   **作用**: 传达品牌价值，吸引用户注意力。
*   **交互**: 包含一个“了解更多”按钮（目前为静态展示）。
*   **样式**: 使用了渐变背景和装饰性背景元素。

### 2.2 游戏卡片网格 (Game Grid)
*   **作用**: 展示所有可用的游戏。
*   **数据源**: 通过 React Query 调用 `gameApi.list()`（后端 `/games` 接口）；保留对 `coverUrl` 与 `lucide-react` 图标的动态映射。
*   **元素**:
    *   **图标**: 动态映射 `lucide-react` 图标 (Moon, Eye, BookOpen)。
    *   **状态标签**: 显示“在线人数”或“敬请期待”。
    *   **Tags**: 游戏类型标签。
    *   **入口按钮**: 点击跳转到 `/game/:id`。

## 3. 开发要点
*   **动态图标渲染**: 使用了一个 `IconMap` 对象将字符串类型的 `coverUrl` 映射为 React 组件。
*   **路由跳转**: 使用 `useNavigate` 进行页面导航。
*   **响应式**: 在移动端为单列，桌面端为三列布局。
