# 注册页 (Register Page)

**文件路径:** `src/pages/Register.tsx`

## 1. 页面概述
新用户创建账号的页面。

## 2. 关键元素
*   **表单区**:
    *   用户名：作为游戏内昵称。
    *   电子邮箱：用于找回密码。
    *   密码 & 确认密码。
*   **协议勾选**: 必须勾选服务条款和隐私政策才能注册。
*   **底部导航**: 跳转到登录页面的链接。

## 3. 开发要点
*   **表单验证**: 使用 HTML5 原生 `required` 属性进行基础验证。
*   **流程**: 当前实现注册成功后直接完成登录并跳转至首页（由 `useAuth.register` 驱动），便于快速体验。
