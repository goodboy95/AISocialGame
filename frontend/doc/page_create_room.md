# 创建房间页 (Create Room Page)

**文件路径:** `src/pages/CreateRoom.tsx`

## 1. 页面概述
这是一个高度动态化的表单页面，用于配置游戏规则。针对狼人杀等复杂游戏，采用了分栏布局。

## 2. 关键元素

### 2.1 布局结构
*   **左侧栏 (基础信息)**:
    *   房间封面、名称、隐私设置（公开/私密）、交流模式（语音/文字）。
*   **右侧栏 (板子配置)**:
    *   **玩家人数**: 按钮组选择。
    *   **板子预设 (Templates)**: 可视化卡片选择（如“预女猎白”、“生推局”）。
    *   **高级规则**: 折叠收纳区，包含女巫规则、胜利条件等详细配置。

### 2.2 底部结算栏
*   **桌面端**: 静态展示在底部。
*   **移动端**: 固定在屏幕底部 (Sticky Footer)，确保在长表单中随时可点击创建。

## 3. 开发要点
*   **Schema 驱动**: 页面依然基于 `game.configSchema` 渲染，但对 `template` 和 `playerCount` 等特定字段进行了特殊 UI 处理（卡片/按钮组），其余字段自动渲染为通用控件。
*   **交互**: 使用 `Collapsible` 组件收纳高级规则，保持界面整洁。
