# 登录页 (Login Page)

**文件路径:** `src/pages/Login.tsx`

## 1. 页面概述
用户进入平台的身份验证入口。采用独立的居中卡片布局，不包含主应用的导航栏。

## 2. 关键元素
*   **Logo 区**: 顶部展示品牌 Logo 和欢迎语。
*   **表单区**:
    *   邮箱/用户名输入框。
    *   密码输入框（带“忘记密码”链接）。
    *   “记住我”复选框。
*   **第三方登录**: 提供 Github 和 Google 快捷登录按钮（目前为 UI 演示）。
*   **底部导航**: 跳转到注册页面的链接。

## 3. 开发要点
*   **布局**: 使用 `min-h-screen flex items-center justify-center` 实现垂直水平居中。
*   **交互**: 提交后调用 `useAuth.login`（底层请求 `/auth/login`），成功即弹出提示并跳转首页；失败展示后端返回的错误信息。
